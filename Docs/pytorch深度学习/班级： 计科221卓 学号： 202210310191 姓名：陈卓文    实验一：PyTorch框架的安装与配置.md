## 1. 实验目的

掌握使用conda创建Python虚拟环境的方法，完成PyTorch深度学习框架及其依赖组件的安装，验证开发环境配置的正确性。

## 2. 实验环境

- 操作系统：Windows/Linux/macOS（需注明具体版本）
- 硬件配置：推荐配备NVIDIA显卡（支持CUDA加速）
- 软件要求：Anaconda/Miniconda（建议2023.10以后版本）

## 3. 实验步骤

### 3.1 创建虚拟环境


`conda create -n pytorch_env python=3.9 # 创建指定Python版本环境 conda activate pytorch_env # 激活虚拟环境`

### 3.2 安装PyTorch

根据硬件配置选择安装命令（访问[PyTorch官网](https://pytorch.org/)获取最新命令）：

`# CUDA 11.8版本（NVIDIA显卡推荐） conda install pytorch torchvision torchaudio pytorch-cuda=11.8 -c pytorch -c nvidia # CPU版本（无独立显卡设备） conda install pytorch torchvision torchaudio cpuonly -c pytorch`

### 3.3 验证安装

`import torch print(f"PyTorch版本: {torch.__version__}") print(f"CUDA可用: {torch.cuda.is_available()}") print("测试张量运算:", torch.rand(3,3) @ torch.rand(3,3))`

## 5. 注意事项

1. 网络环境：确保conda源配置正确（建议使用国内镜像源加速下载）
2. 版本匹配：CUDA版本需与显卡驱动兼容
3. 存储空间：预留至少3GB磁盘空间用于安装依赖
4. 环境管理：实验结束后可使用`conda deactivate`退出虚拟环境
