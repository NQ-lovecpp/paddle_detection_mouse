### Python 列表

**列表** 是Python中一种内置的数据结构，用于存储多个有序的元素。列表是可变的（mutable），这意味着可以动态地修改列表中的元素。列表中的元素可以是任何类型，包括数字、字符串、甚至其他列表。

#### 列表的创建与操作

1. **创建列表**
   ```python
   my_list = [1, 2, 3, 4, 5]
   ```
   
2. **访问元素**
   ```python
   print(my_list[0])  # 输出 1
   print(my_list[2])  # 输出 3
   ```

3. **修改元素**
   ```python
   my_list[1] = 10
   print(my_list)  # 输出 [1, 10, 3, 4, 5]
   ```

4. **添加元素**
   ```python
   my_list.append(6)
   print(my_list)  # 输出 [1, 10, 3, 4, 5, 6]
   ```

5. **删除元素**
   ```python
   del my_list[1]
   print(my_list)  # 输出 [1, 3, 4, 5, 6]
   ```

6. **嵌套列表**
   ```python
   nested_list = [1, [2, 3], [4, 5]]
   print(nested_list[1])  # 输出 [2, 3]
   print(nested_list[1][1])  # 输出 3
   ```

### 列表切片

**列表切片** 是一种通过指定索引范围来访问列表中一个子集的方法。切片通过使用冒号 `:` 语法来实现，可以指定起始位置、结束位置和步长。

#### 切片的语法
```python
list[start:stop:step]
```

- **start**: 切片开始的位置（包含该位置，默认为 0）。
- **stop**: 切片结束的位置（不包含该位置，默认为列表长度）。
- **step**: 切片的步长（默认为 1）。

#### 切片示例

1. **基本切片**
   ```python
   my_list = [1, 2, 3, 4, 5]
   sub_list = my_list[1:4]
   print(sub_list)  # 输出 [2, 3, 4]
   ```

2. **省略 start 和 stop**
   ```python
   sub_list = my_list[:3]  # 等价于 my_list[0:3]
   print(sub_list)  # 输出 [1, 2, 3]
   
   sub_list = my_list[2:]  # 等价于 my_list[2:len(my_list)]
   print(sub_list)  # 输出 [3, 4, 5]
   ```

3. **使用负索引**
   ```python
   sub_list = my_list[-4:-1]
   print(sub_list)  # 输出 [2, 3, 4]
   ```

4. **指定步长**
   ```python
   sub_list = my_list[0:5:2]
   print(sub_list)  # 输出 [1, 3, 5]
   ```

5. **反向切片**
   ```python
   sub_list = my_list[::-1]
   print(sub_list)  # 输出 [5, 4, 3, 2, 1]
   ```

### 总结

- **列表** 是Python中用来存储多个有序元素的数据结构，具有动态调整大小的特点。
- **列表切片** 允许通过指定索引范围和步长来访问列表中的子集，是一种灵活而强大的数据访问方式。
