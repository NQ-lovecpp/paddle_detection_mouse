### 常见运算符的比较

1. **算术运算符**：
   - Python 和 C++ 都支持基本的算术运算符如加（`+`）、减（`-`）、乘（`*`）、除（`/`）等。
   - Python 有一个特殊的整数除运算符（`//`），用于得到整数除法的商，而在 C++ 中，整数除法直接使用 `/` 即可。
   - Python 使用 `**` 进行幂运算，相比之下，C++ 需要使用 `pow()` 函数或复杂的表达式。

2. **比较运算符**：
   - Python 的比较运算符（`<`, `>`, `==`, `<=`, `>=`, `!=`）与 C++ 大致相同。
   - Python 允许链式比较，如 `a < b < c`，这在 C++ 中需要分开写成 `a < b && b < c`。

3. **逻辑运算符**：
   - Python 使用 `and`、`or` 和 `not`，而 C++ 使用 `&&`、`||` 和 `!`。这是一个常见的初学者错误源。

4. **位运算符**：
   - Python 和 C++ 都支持位运算符，如位与（`&`）、位或（`|`）、位非（`~`）、位异或（`^`）、左移（`<<`）和右移（`>>`）。

5. **赋值运算符**：
   - Python 支持 C++ 中的所有基本赋值运算符，如 `=`, `+=`, `-=`, `*=`, `/=`, `%=` 等。
   - Python 还支持 `//=`, `**=` 和 `@=`（用于矩阵乘法）。

6. **身份运算符**：
   - Python 引入了 `is` 和 `is not` 运算符，用于测试两个变量是否引用同一个对象，这是 C++ 中没有的。
   - 这与 C++ 中的地址比较 `&a == &b` 类似，但在 Python 中更加直观和常用。

7. **成员运算符**：
   - Python 使用 `in` 和 `not in` 检查一个元素是否存在于序列中，这种语法在 C++ 中通常需要调用方法或使用算法。

### 特殊注意点

- **除法操作**：Python 3 中，除法运算符 `/` 总是执行真除法（即使操作数都是整数），返回一个浮点数。如果需要执行**地板除以**得到整数结果，应使用 `//`。
- **增强赋值的行为**：在处理可变数据类型（如列表）时，增强赋值（如 `+=`）可能会就地修改原有数据，这一行为在 Python 中比 C++ 更加常见和重要。

总的来说，虽然很多基础运算符在 Python 和 C++ 中表现类似，但 Python 在逻辑运算符和一些特殊运算符（如身份运算符和成员运算符）上的不同，需要特别注意和适应。这些运算符的使用简化了代码的编写，也是 Python 易于编程的原因之一。
